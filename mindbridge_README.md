## Module 1 — Synthetic Data Generator

### What it generates
- **500 student profiles** across 3 mental health trajectories
- **56,000 daily behavioral records** (500 students × 112 days)
- **8,000 weekly mood check-ins** with realistic 15% non-response
- **8,000 risk labels** (Low / Medium / High) for ML training

### Trajectories
| Trajectory | % of Students | Pattern |
|---|---|---|
| Healthy | 50% | Stable signals all semester |
| Declining | 30% | Gradual drift starting mid-semester (Week 4-9) |
| Crisis | 20% | Severe deterioration late semester (Week 8-13) |

### Behavioral Signals Generated
| Signal | Description | Healthy Baseline |
|---|---|---|
| sleep_hours | Hours slept per night | 7.3–8.2 hrs |
| bedtime_hour | Hour of sleep (24h) | 23:00–23:30 |
| lms_logins | Daily LMS login count | 4.2 weekdays |
| study_hours | Study session hours | 3.8 hrs/day |
| social_score | Social engagement 0-10 | 7.2 |
| dining_visits | Dining hall visits/day | 2.5 |
| assignment_delta | Days before deadline | +1.8 days early |
| mood_score | Weekly self-report 1-10 | 7.5 |
